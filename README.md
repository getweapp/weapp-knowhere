# 微信小程序－知亦行

### 说明：

实现了知亦行首页功能，特色：
- 顶部tab栏选择
- 上拉加载
- 较美观的图文排版

### 数据接口:

使用本地数据

### 目录结构：

- images — 存放项目图片文件
- pages — 存放项目页面渲染相关文件
- style — 存放公共样式库
- utils — 存放数据获取，全局变量代码

### 开发环境：

微信web开发者工具 v0.11.112301
